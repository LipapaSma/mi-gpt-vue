# MiGPT 可视化配置界面

基于 Vue 的 MiGPT 可视化配置界面，提供直观的服务管理和参数配置能力。

## 🌟 特性

- 💻 支持 PC 端和移动端自适应布局
- 🎨 清晰直观的配置界面
- 🚀 一键启动/停止服务
- ⚙️ 可视化参数配置
- 🔄 实时状态监控

## 💡 使用提示

1. 确保系统环境已正确配置
2. 访问配置界面进行参数设置
3. 使用启动按钮开启服务
4. 可随时查看服务状态
5. 需要时可一键停止服务
